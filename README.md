# Fake News Detector

This project is a Fake News Detection system built using Python and Machine Learning. It analyzes news text and predicts whether it is REAL or FAKE using Natural Language Processing (NLP) techniques.

## Features
- Text preprocessing
- Machine Learning classification
- Predicts REAL or FAKE news
- Built with Python, Pandas, and Scikit-learn

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (Text Vectorization)

This project was created to practice Machine Learning and NLP concepts while building a real-world application.
